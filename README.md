# Панель администратора проекта "Need Car Speed"

Используемые технологии: 
* React, 
* Redux.
---
Препроцессоры: 
* SCSS.
---
Дополнительные библиотеки: 
* prop-types, 
* axios.

# Возможности: 
1. Создание и редактирование сущностей системы: 
---
* `Автомобили`
* `Заказы`
* `Пункты выдачи`
* `Города`
---
2. Авторизация: 
* `JWT`
---

# Также имеется:
